Airline Tweets Sentiment Analysis Project

Files Included:
1. ML - Technical report containing:
   - Problem introduction
   - Dataset characteristics
   - Model selection and methodology
   - Results and analysis
   - Conclusion and references

2. .ipynb - Jupyter Notebook with complete Python code used for:
   - Data loading
   - Preprocessing
   - Model training (SVM, Weighted SVM with n-grams, Neural Network)
   - Evaluation and visualizations

3. tweets.csv - Dataset containing US airline tweets labelled as positive, negative, or neutral.

Instructions to Run:
1. Place 'tweets.csv' in the same directory as the .ipynb.
2. Open the .ipynb in Jupyter Notebook or Google Colab.
3. Run all cells in order from top to bottom to reproduce the results in the report.

Requirements:
- pandas
- numpy
- scikit-learn
- matplotlib
- nltk

Note:
The dataset 'tweets.csv' is the publicly available US Airline Sentiment dataset.
